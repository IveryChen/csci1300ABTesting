Part 2: Analysis

Hypothesis

1. Misclick Rate

the frequency with which users click something else on the page before finding the correct button for the task

Null hypothesis (H0): there is no difference between in misclick rates between versions A and B of the page

Alternative Hypothesis (H1):there is a difference between misclick rates between versions A and B, with higher misclick rate in A than in B

Prediction: I anticipate rejecting my null hypothesis because Page B and Page A show significant differences in the user interface that it should impact user patterns

Reasoning behind the alternative hypothesis: Page B is designed to minimize misclicks by highlighting certain information and separating information, suggesting it should be more effective in reducing misclick rates compared to Page A

2. Time on Page

time spent on the webpage for each user group

Null hypothesis (H0): The average time spent on the webpage is the same for both versions A and B of the page

Alternative Hypothesis (H1): The average time spent on the webpage varies across both versions A and B, where users spend more time on A than B

Prediction: I anticipate to reject my null hypothesis because Page B and Page A are different enough for users to see information differently and therefore leading to reduced time spent on webpage

Reasoning behind the alternative hypothesis: Page B highlights more actionable buttons which will reduce the amount of time users spend on finding them

3. Appointment Completion Rate

percentage of users who successfully book an appointment with the correct doctor, at the correct location and at the desired time

Null hypothesis (H0): There is no difference in appointment completion rates between Versions A and B of the page

Alternative Hypothesis (H1): There is a significant difference in appointment completion rate between versions A and B, where users have a higher appointment completion rate on B than A

Prediction: I anticipate to fail to reject my null hypothesis because eventually all users will be able to complete the appointment

Reasoning behind the alternative hypothesis: Page B highlights more actionable buttons which will reduce the amount of time users spend on finding them

Statistical Tests

1. Misclick Rate

I chose chi-squared test because misclicking is a categorical variable (i.e., users either misclick or they don’t). The chi-squared statistic (χ²) is 10.08, with a p-value of 0.0015 and degrees of freedom (df) of 1. These values indicate that the observed difference in misclick rates between versions A and B is statistically significant. The p-value being lower than the chosen significance level of 0.05 suggests strong evidence against the null hypothesis. Therefore, I reject the null hypothesis. This supports the alternative hypothesis, which posited that version B, designed to reduce misclicks, would have a lower misclick rate compared to version A.

2. Time on Page

I chose one-tailed t-test because I’m comparing the means of two independent groups (users on version A and users on version B). The t-test resulted in a t-score of 9.2207 and a p-value of 0.9999999991. These values suggest that the observed difference in the average time spent on the webpage between versions A and B is statistically significant. Since the p-value is very close to 1, the probability of observing such a large difference in average time spent on the webpage by chance is extremely low. This suggests strong evidence against the null hypothesis. Therefore, I reject the null hypothesis. This result supports the alternative hypothesis that version B, with its highlighted buttons, would lead to reduced time spent on the webpage compared to version A.

3. Appointment Completion Rate

I chose chi-squared test because appointment completion rate is a categorical variable (users either complete the appointment or they don't). The chi-squared test value (χ²) is 1.0213, with a p-value of 0.3122. These values indicate that the observed difference in appointment completion rates between versions A and B is not statistically significant. With the p-value being greater than the chosen significance level of 0.05, there is insufficient evidence to reject the null hypothesis. Therefore, I fail to reject the null hypothesis. This result aligns with the prediction that eventually, all users will be able to complete the appointment regardless of the version they interact with.
